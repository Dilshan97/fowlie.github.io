# fowlie.github.io
My personal blog made with [Jekyll](https://jekyllrb.com) and hosted on [GitHub Pages](https://pages.github.com).

### Development setup guide
  1. sudo apt-get install ruby-dev
  2. sudo gem install jekyll bundler
  3. bundle install
  4. jekyll serve
